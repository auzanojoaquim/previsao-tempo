# Previsão do Tempo 🌦️

Este é um projeto simples em **JavaScript** para consultar a previsão do tempo.

## 🚀 Como usar
1. Entra no site.
2. Digite o nome da cidade.
3. Veja as condições atuais do tempo.

## 🔧 Tecnologias
- HTML
- CSS
- JavaScript (API de clima)
